# Как запустить проект:

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/f1sher1762/docka.git
   cd docka
   
2. Запустите контейнеры:
docker-compose up -d


3. Доступ к WordPress будет по адресу http://localhost:8080, а к phpMyAdmin — http://localhost:8081.


4. CI/CD: Процесс CI/CD настроен с использованием GitHub Actions. При каждом изменении в ветке main автоматически выполняется коммит и пуш изменений обратно в репозиторий. Это позволяет поддерживать актуальность кода и автоматически обновлять проект без необходимости вручную выполнять эти шаги.

ps. Подробнее о конфигурации CI/CD:

Включает автоматический пуш изменений в репозиторий.
Использует секреты для безопасного хранения данных.
Для работы с GitHub используется SSH-ключ для аутентификации
